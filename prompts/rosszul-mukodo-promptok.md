# Rosszul működő promptok

Az alábbi promptok kevésbé működtek jól, mivel nem tartalmaztak elegendő kontextust vagy túl általánosak voltak.

---

## 1. Dashboard hiba

**Prompt:**
"Admin Dashboard is not loading correctly..."  

**Probléma:**
- nincs kódrészlet
- nem derül ki a környezet
- nehezen reprodukálható

**Javítás:**
- konkrét komponens kód megadása
- API hívás bemutatása

---

## 2. Loading state hiba

**Prompt:**
"A loadStats metódusban leragad..."  

**Probléma:**
- hiányzó implementáció
- nincs stack trace
- nincs async flow bemutatva

---

## 3. UI probléma

**Prompt:**
"The 'Ár módosítása' button not doing anything..."  

**Probléma:**
- több probléma egy promptban
- nincs konkrét komponens
- nincs eseménykezelés bemutatva

---

## 4. Számítási hiba

**Prompt:**
"A végösszeg nem frissül..."  

**Probléma:**
- nincs adatmodell
- nincs számítási logika

---

## Összegzés

A rossz promptok jellemzői:
- túl általánosak
- hiányzik a kontextus
- nem tartalmaznak konkrét példát

**Tanulság:**
Minél részletesebb és konkrétabb egy prompt, annál jobb választ ad az AI.
