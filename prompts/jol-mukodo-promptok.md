# Jól működő promptok

Az alábbi promptok jól működtek, mivel konkrétak, egyértelműek és megfelelő kontextust biztosítottak.

---

## 1. Unit tesztek generálása

**Prompt:**
"Create Unit tests for AuthController with NUnit."  

**Miért jó:**
- konkrét cél
- technológia megadva (NUnit)
- pontos scope

---

## 2. Service implementáció

**Prompt:**
"Készíts egy GridFSService osztályt .NET 10-ben..."  

**Miért jó:**
- részletes követelmények
- metódusok pontos leírása
- technológiai stack megadva

---

## 3. CI/CD pipeline

**Prompt:**
"Update the GitHub Actions CI workflow..."  

**Miért jó:**
- teljes projektstruktúra megadva
- konkrét verziók
- pontos elvárások

---

## 4. Feature fejlesztés

**Prompt:**
"Create an admin feature, where administrator can inactivate user's account."  

**Miért jó:**
- jól definiált funkció
- rendszer szintű gondolkodást igényel
- backend + frontend érintett

---

## 5. Docker konfiguráció

**Prompt:**
"Create docker files for the project."  

**Miért jó:**
- egyértelmű cél
- projekt szintű feladat

---

## Összegzés

A jól működő promptok jellemzői:
- konkrétak
- tartalmaznak technológiai kontextust
- egyértelmű elvárásokat fogalmaznak meg
